# Basic-Calculator-v0.3

A simple console-based calculator written in Python.

## Features
- Supports addition, subtraction, multiplication, and division.
- Optional **Random Mode**:
  - If you choose an invalid operation or enter invalid numbers, the program will randomly pick the operation and/or numbers for you.
- Handles invalid inputs and division by zero safely.

## How to Run
1. Make sure you have Python installed.
2. Run the script:
   ```bash
   python basic_calculator_v0_3.py
